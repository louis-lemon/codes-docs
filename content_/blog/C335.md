## Overview

This is a simple guide for applying and deploying the chat service on EurekaCode in just 5 minutes.

You can get started quickly by using the default settings.



##  Quick Start (5 Steps) 



### Step 1: Log In

- Go to the EurekaCodes homepage
- Click the social login button (Google)
- Complete the account linking
![ENG image](https://image.lemoncloud.io/ce2eaa41-6110-4775-a51e-7e14cf93644f)



### Step 2: Check the Default Workspace

- A default workspace is automatically created when you log in
- You can use it immediately without any additional setup
- Workspace name:default
![ENG image](https://image.lemoncloud.io/b4aac70a-98b7-4ee6-90bd-4274fad4063f)



### Step 3: Create a Project

- Click the "Add Project" button
![ENG image](https://image.lemoncloud.io/f9aaec70-f09b-4c36-ac5e-c8ae4ee9ab0e)

- Configure your project settings
  - Project name: my-chat-app (example)
  - Cloud: Free Plan AWS
  - Region: Asia Pacific (Seoul)
  - Stage: Develop
- Click the "Create" button
![ENG image](https://image.lemoncloud.io/af1ae576-2902-4467-9195-5ba871aa1981)



### Step 4: Apply for Chat Service

- Click the "Add Service" button
![ENG image](https://image.lemoncloud.io/71fa7c43-a462-4c97-bda3-90af5ac230a8)

- Select eureka-chat-front from the catalog list
![ENG image](https://image.lemoncloud.io/3716fced-3e45-4056-aa00-830b87fd55a7)

- Click the "Apply for Service" button
![ENG image](https://image.lemoncloud.io/58dd0d44-5692-4990-bef8-7892124c2875)

- Agree to the terms and conditions → Application completed


### Step 5: Automatic Deployment and Usage

- Admin approval and deployment are processed automatically
![ENG image](https://image.lemoncloud.io/ed86e63c-445c-4681-a8f7-bcaebb0e2221)

- Wait for deployment to complete (about 3–5 minutes)
- Deployment is finished
- Access the chat service using the provided URL
![ENG image](https://image.lemoncloud.io/b63531a6-d1fc-4722-9797-ba38b977cc96)
