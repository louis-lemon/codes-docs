## 개요

유레카코즈에서 채팅 서비스를 **5분 만에** 신청하고 배포하는 간단한 가이드입니다.

기본 설정만 사용하여 빠르게 시작할 수 있습니다.



##  빠른 시작 (5단계)



### 1단계: 로그인

- EurekaCodes 홈페이지 접속
- 소셜 로그인 버튼 클릭 (Google)
- 계정 연동 완료
![유레카코즈 퀵 스타트 가이드 with 채팅앱 image](https://image.lemoncloud.io/ce2eaa41-6110-4775-a51e-7e14cf93644f)



### 2단계: 기본 워크스페이스 확인

- 로그인 시 기본 워크스페이스가 자동 생성됨
- 추가 설정 없이 바로 사용 가능
- 워크스페이스 이름: `default`
![유레카코즈 퀵 스타트 가이드 with 채팅앱 image](https://image.lemoncloud.io/b4aac70a-98b7-4ee6-90bd-4274fad4063f)



### 3단계: 프로젝트 생성

- 프로젝트 추가 버튼 클릭
![유레카코즈 퀵 스타트 가이드 with 채팅앱 image](https://image.lemoncloud.io/f9aaec70-f09b-4c36-ac5e-c8ae4ee9ab0e)

- 프로젝트 설정
  - 프로젝트명: `my-chat-app` (예시)
  - 클라우드: `Free Plan AWS`
  - 리전: `Asia Pacific (Seoul)`
  - 스테이지: Develop
- 생성 버튼 클릭
![유레카코즈 퀵 스타트 가이드 with 채팅앱 image](https://image.lemoncloud.io/af1ae576-2902-4467-9195-5ba871aa1981)



### 4단계: 채팅 서비스 신청

- 서비스 추가 버튼 클릭
![유레카코즈 퀵 스타트 가이드 with 채팅앱 image](https://image.lemoncloud.io/71fa7c43-a462-4c97-bda3-90af5ac230a8)

- 카탈로그 목록에서 `eureka-chat-front` 선택
![유레카코즈 퀵 스타트 가이드 with 채팅앱 image](https://image.lemoncloud.io/3716fced-3e45-4056-aa00-830b87fd55a7)

- 서비스 신청 버튼 클릭
![유레카코즈 퀵 스타트 가이드 with 채팅앱 image](https://image.lemoncloud.io/58dd0d44-5692-4990-bef8-7892124c2875)

- 이용 약관 동의 → 신청 완료
