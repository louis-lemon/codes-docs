# Eureka Codes User Guide: Using the Pro Plan

### 1. Login

![ENG image](https://image.lemoncloud.io/328fcf6a-8f59-451a-bb7d-ceed231716a4?width=1024)

* Please log in with your Google account.

***

### 2. Getting Started with Workspace

![ENG image](https://image.lemoncloud.io/6caacd84-7db1-40f2-83d8-5f4aad670e11)

* Workspaces serve as the basic unit for managing your projects.
* When you log in, a default workspace is already created for you, so you can start using it right away without any additional setup
* If you'd like to add a new workspace, you can simply click the "Add Workspace" button on the right side.

***

### 3. Create a New Workspace

![ENG image](https://image.lemoncloud.io/bd1ebd73-5152-4fff-a26e-bdce508a65d3?width=1024)

* There are three types of workspace plans: Basic, Pro, and Advanced.

![ENG image](https://image.lemoncloud.io/62cdfdd0-3a61-4a95-bbfd-d2a7f8d4ba97)

* Please add a payment method before creating a Pro Plan workspace.
* When you click the "Add Card" button, it will take you to the card registration screen.

![ENG image](https://image.lemoncloud.io/9569c9d7-422c-48c4-b2ac-0c6c9301ca1f)

* Enter your email, card number, and other information to complete the registration.

![ENG image](https://image.lemoncloud.io/8b807883-72e3-41cf-8e6b-e1a4261cce62)

* Go back to the workspace registration screen and enter the required information.
* You need to specify the workspace name and AWS region, and email verification is required.

![ENG image](https://image.lemoncloud.io/a1b73197-c331-4fb4-b7aa-f2d8f7801656)

* Please verify your email address (a valid email is required).

![ENG image](https://image.lemoncloud.io/cbabf253-cb09-41e7-a207-1752bc2e9259)

* After you complete email verification, please select a region.

![ENG image](https://image.lemoncloud.io/3418fb0f-750e-4ad9-a79f-e5272f760abd)

* All the information must be entered, and if you select "Agree to the Terms of Service", the button below will be enabled.
* Click the activated button to complete the workspace creation.

![ENG image](https://image.lemoncloud.io/720d9b18-7bc3-4a3a-8ef5-f36d4c0d097d)

* Please check the workspace information you created.

***

### 4. Create Project

![ENG image](https://image.lemoncloud.io/d9c12256-d401-4647-9b8e-bf6a47e4899a)

* To use the service, please create a project.
* Please click the "Add Project" button.

![ENG image](https://image.lemoncloud.io/0e11e8b1-a368-4d8c-af38-9a6672718f8a)

* Check the details entered during workspace creation, choose a stage (development or production), and proceed to create your project.

![ENG image](https://image.lemoncloud.io/1c240d40-87e4-4ec1-9fa5-f05a354e8dbc)

* You can request the services you need from the catalog list.

***
