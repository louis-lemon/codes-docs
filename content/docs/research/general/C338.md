###  LangChain

- 대규모 언어 모델(LLM)을 기반으로 애플리케이션을 개발하기 위한 프레임워크입니다.
- 특징
  - 다양한 LLM, Tool, Memory를 체인 형태로 연결합니다.
  - 외부 데이터 소스와의 통합을 지원합니다.
  - 정해진 워크플로의 순서대로 실행되는 선형적인 체인 구조입니다.
- 기존 LLM과의 차이점
  - 기존 LLM은 텍스트 생성과 질문 및 답변에 중점적이지만, LangChain은 외부 데이터와 통합하여 복잡하고 실용적인 애플리케이션 구축이 가능합니다.


### LangGraph

- 그래프 구조를 활용하여 유연하고 동적인 워크플로를 구현할 수 있는 프레임워크 입니다.
- 핵심 구성 요소
  - 노드(Node)
    - 워크플로에서 실행되는 개별 작업 단위입니다.
    - LLM 호출, Tool 사용, 데이터 처리 등의 기능을 실행할 수 있습니다.
  - 엣지(Edge)
    - 노드간의 연결을 의미하며, 작업 흐름을 결정합니다.
  - 상태(State)
    - 모든 노드가 공유하는 데이터 저장소입니다.
    - 여러 에이전트는 state를 통해 정보를 공유하고 워크플로의 문맥을 유지합니다.
  - 에이전트(Agent)
    - LLM이 어떤 노드를 실행할지 판단하는 '에이전트' 역할을 수행할 수 있습니다.
- 특징
  - 내부적으로 state를 추적하고 적절한 MCP를 자유롭게 호출할 수 있습니다.
